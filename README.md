# Student Result System

I. Project Description
This project implements a Student Result System using MySQL to manage student marks across subjects and perform analytical queries.

II. Technologies Used
- MySQL

III. Features
- Multi-table relationships (Students, Subjects, Marks)
- Aggregate functions (AVG, MAX)
- GROUP BY & HAVING
- Subqueries for top performer identification
- Subject-wise highest marks calculation
